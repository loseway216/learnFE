## nagivation cheatsheet

### 原生

- `%` Nearest/matching {[()]}
- `[(` `[{` `[<` Previous ( or { or <
- `]` Next
- `[m` Previous method start
- `[M` Previous method end

### vim-surround

- `cs` change surround
- `ds` delete surround
- `ysiw` surround word
- `yss` surround whole line
- v/V + S surround 选中部分

### vim-easymotion

- `<Leader>s` search
- `<Leader>f?` 聚焦目标字母
- `<Leader>w` 往下聚焦单词
- `<Leader>b` 往上聚焦单词
- `<Leader>j` 聚焦下面行首
- `<Leader>k` 聚焦上面行首
